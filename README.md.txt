# 🌍 Planejador de Viagem com IA

Este aplicativo interativo cria roteiros de viagem personalizados com ajuda de agentes inteligentes usando a biblioteca [CrewAI](https://docs.crewai.com/).

## ✈️ Funcionalidades

- Geração de roteiros de viagem personalizados com base em destino, número de dias e interesses.
- Uso de múltiplos agentes (planejador, pesquisador e redator) para criar um plano completo.
- Visualização de mapa interativo do destino com Folium.
- Interface amigável via Streamlit.
- Opção para baixar o roteiro como arquivo `.txt`.

## 🧪 Tecnologias Utilizadas

- [Python](https://www.python.org/)
- [Streamlit](https://streamlit.io/)
- [CrewAI](https://github.com/joaomdmoura/crewAI)
- [LangChain](https://www.langchain.com/)
- [OpenAI API](https://platform.openai.com/)
- [Folium](https://python-visualization.github.io/folium/)
